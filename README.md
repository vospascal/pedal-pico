# pedal-arduino
it reads 0 - 5 volt scale. So as long as its within that range it'll work.

Measure the current, use "Arduino Micro Pro" as to read the analogue value from the sensor

## conectors pedal side
1 V5/vcc
5 GND
6 analog/a1

2 V5/vcc
3 GND
4 analog/a1


## one loadcell example

| hx711  | arduino  | 
|---|---|
| gnd  |  gnd |
| sck  | pin 5  | 
| vcc  | vcc  |
| dt(data)  |  pin 7 |

## multi loadcell example

| hx711  | arduino  | 
|---|---|
| gnd  |  gnd |
| vcc  | vcc  |
| sck(Clock) - Thr  | pin 9  |
| sck(Clock) - Brk | pin 7  |
| sck(Clock) - Clt | pin 5  |
| dt(Data) - Thr |  pin 8 |
| dt(Data) - Brk |  pin 6 |
| dt(Data) - Clt |  pin 4 |
